## Commands
Here are all the available commands and what they do:
|       Command         |               Usage               |
|---------------------- | --------------------------------- |
| `cat [file]`          | Open a link from `ls`.            |
| `cheatcode [code]`*   | Enter a cheat code.               |
| `clear`               | Clears answer.                    |
| `echo [string]`       | Prints a string.                  |
| `exit`                | Returns to home.                  |
| `help`                | Shows this page.                  |
| `ls`                  | Lists all links on the website.   |
| `reboot`              | Reloads the page.                 |

*There are many cheat codes on this website. You could look through the source code to find them, or you could try a few things!
